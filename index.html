Basic Select Queries

Retrieve data from your tables with simple select statements.

select * from table1
select a, b from table1

Aggregate Functions

Use standard SQL aggregate functions to analyze your data.

select max(a), min(b) from table1
select avg(price), sum(quantity), count(*) from orders

Distinct Values

Find unique values in your dataset columns.

select distinct(a) from table1
select distinct(category) from products

Conditional Queries

Filter your data with WHERE clauses and conditions.

select a from table1 where a = 10
select a, b from table1 where a = c or b = 5

Table Joins

Combine data from multiple tables for comprehensive analysis.

select * from table1, table2

Table Aliasing

Use table aliases for more readable and efficient queries.

select a, t1.b, c, t2.d from table1 as t1, table2 as t2 where t1.b = t2.b and t2.d >= 0
